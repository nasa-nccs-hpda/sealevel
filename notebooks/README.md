
# sealevel/notebooks directory

- Python project to orchestrate modules within the Framework for Accessing Changes To Sea-level (FACTS).  

## Objectives

- These Notebooks run in Discover jupyter hub:  https://jh-discover.nccs.nasa.gov.  Execute in order:
1. [Runs select FACTS modules (e.g., fair, lws, and stereodynamics) ](https://github.com/nasa-nccs-hpda/sealevel/blob/main/notebooks/fair_facts_v2.ipynb)
2. [Aggregates total sealevel deltas from  #1 output ](https://github.com/nasa-nccs-hpda/sealevel/blob/main/notebooks/fair_facts_v2_total.ipynb)
3. [Visualizes total sealevel deltas from #2 output ](https://github.com/nasa-nccs-hpda/sealevel/blob/main/notebooks/fair_facts_v2_total.ipynb)
